プロフィール

作業が遅くて悩んでいた自分が、AIを取り入れたことで仕事の効率がグッと上がりました。

スキルセット

  • プロンプト設計（業務効率化・FAQ設計向け）
  • Dify / Udify（チャットボット・ワークフロー構築）
  • cursor（AIコーディング環境）
  • ChatGPT API / Claude / Gemini（API連携による自動化）
  • Midjourney（画像生成・プロトタイピング用途）
  • Google Apps Script（Google Workspace連携による業務自動化）
