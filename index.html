プロフィール

1993年生まれ・大阪在住。
業務効率化に特化したAIツールの導入支援や仕組みづくりを行っています。

ITエンジニアとして約7年間、開発・運用・業務改善に携わった経験を活かし、
現場に合わせたわかりやすい仕組みづくりを得意としています。

「現場で実際に使える仕組み」を意識した導入支援を心がけ、
課題のヒアリング〜仕組み設計〜構築・運用サポートまで一貫して対応しています。

「手作業が多くて時間が足りない」「社内の問い合わせ対応が煩雑」など、
日々の業務の課題を、AIの力を借りてスムーズに解決できるようサポートします。

対応業務

  • プロンプト設計（例：社内FAQの自動応答、一次対応ボット）
  • Dify / Udify：チャットボットや業務ワークフローの構築
  • ChatGPT API / Claude / Gemini：社内業務自動化への応用
  • Google Apps Script（GAS）：スプレッドシート連携で集計・日報作成
  • Midjourney：画像生成を使ったプロトタイプ資料作成
  • cursor：AI活用環境での軽量なツール開発・改修、LPなどのシンプルなWebサイト制作

よくある疑問に答える、LINEチャットボット

原状回復に関する問い合わせ対応を、LINE上で完結できるように構築したチャットボット
国交省のガイドラインに準拠しながら、実務に即したQ&Aを設計。

経費精算・就業規則チャットフロー

「この場合ってどうなるんだっけ？」という社内の細かい質問に、チャットで即対応。
担当者の代わりとなるフローを設計し、業務の流れを止めずにルール運用をサポートします。
情報の分散や属人化を防ぎ、チーム全体で使えるナレッジへと変換しています。

議事録ワークフロー

会議の音声やメモをもとに、要点を抽出して自動で議事録を整形。
書き起こしだけでなく、構成や読みやすさにも配慮し、即使えるドキュメントに仕上げます。
「書く時間がない」「整理が面倒」といった課題を、AIでまるごと解決します。

領収書自動管理ツール

PDFや紙の領収書から、日付・金額・発行元などの情報を自動で抽出・分類。
フォーマット違いやばらつきにも柔軟に対応し、煩雑だった経費整理の負担を軽減します。
正確性と手間削減を両立した、実務向けのスマートなツールです。
